## 💡 The Main Feature: AUTOMATIC SALARY CALCULATION

### What It Does
```
Employee works 09:00 - 17:30 with hourly rate €15/hour
System AUTOMATICALLY calculates:
  Hours: 8.5h
  Daily Salary: €127.50
  
NO MANUAL CALCULATIONS NEEDED! ✓
```

### How It Works
- Employee records check-in/check-out time
- System calculates: (Check-out - Check-in) = Hours
- System calculates: Hours × Hourly Rate = Daily Salary
- Result displayed instantly in table
- Monthly totals calculated automatically

## 💰 Salary Calculation Examples

### Example 1: Standard 8-Hour Day
```
Check-in: 09:00
Check-out: 17:00
Hourly Rate: €20/hour
Result: 8.0 hours = €160.00 daily salary
```

### Example 2: 8.5-Hour Day
```
Check-in: 09:00
Check-out: 17:30
Hourly Rate: €15/hour
Result: 8.5 hours = €127.50 daily salary
```

### Example 3: Monthly
```
22 working days × 8 hours/day = 176 hours
176 hours × €15/hour = €2,640 gross
Minus deductions = €2,376 net
All calculated automatically!
```
